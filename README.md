# First approach on python openCV and computer vision in general

## Requirement

* Python 3.10
* pip version 23.1.2
* opencv-python
* mediapipe
* pyautogui

## Installation
* Clone this project
* In terminal, run:
```
pip install opencv-python
pip install mediapipe
pip install pyautogui
```

## Run
* In the terminal, cd to this project's folder and run
```
python.exe main.py
```
